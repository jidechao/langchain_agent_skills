# Skills Agent Web UI

React + TypeScript chat UI for the LangChain Skills Agent.

## Development

```bash
npm install
npm run dev
```

By default the frontend calls `http://localhost:8000`.

To override:

```bash
VITE_API_BASE_URL=http://127.0.0.1:8000 npm run dev
```

## Commands

```bash
npm run test
npm run lint
npm run build
```
